

# Курсовой проект: игра Color Lines

## Описание

Игра проходит на поле 9x9. На поле появляются цветные шарики, всего 7 цветов. Игрок может перемещать шарик по горизонтали или вертикали, если путь свободен. Если собирается линия из 5 или более шариков одного цвета, эта линия удаляется, а игрок получает очки.

## Возможности

- поле 9x9
- 7 цветов шариков
- добавление 3 новых шариков после хода
- показ следующих шариков
- удаление линий по горизонтали, вертикали и диагонали
- подсчёт очков
- 3 уровня сложности
    - лёгкий
    - средний
    - сложный

## Уровни сложности

### Лёгкий уровень

Компьютер анализирует поле и пытается помочь игроку.  
Если на поле есть линия из 3 или 4 шариков и среди следующих шариков есть подходящий цвет, компьютер ставит шарик в продолжение этой линии.

### Средний уровень

Компьютер ставит новые шарики случайно.

### Сложный уровень

Компьютер анализирует поле и пытается помешать игроку.  
Если есть линия из 3 или 4 шариков, компьютер через ход пытается поставить в продолжение линии шарик другого цвета.
